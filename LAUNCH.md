## ЧТО СТРОИТ КОНТУР (пошаговый план реализации)

Двигайся этапами, каждый — коммит в GitHub + отчёт в Telegram. Каждый этап оставляет проверяемый публичный артефакт.

**Этап 1 — Фундамент (документы + репо):**

- Перенести манифест/конституцию (из этого файла) в `/docs` как отдельные открытые markdown-документы.
- README с миссией, принципами, статусом, дисклеймером (НЕ инвестиция, НЕ пирамида).
- Лицензия Open Source (напр. MIT/AGPL для кода + CC-BY-SA для документов).

**Этап 2 — Публичный сайт (GitHub Pages):**
- Простой открытый сайт: миссия, конституция, направления помощи, прозрачность, как участвовать.
- Деплой через GitHub Pages (или Cloudflare Pages). Публичная ссылка в отчёт.

**Этап 3 — Прозрачность данных:**
- IPFS-пиннинг ключевых документов (конституция, отчёты) — неизменяемые записи.
- Публичные реестры решений/операций.

**Этап 4 — Управление (governance), TESTNET:**
- Snapshot space (офф-чейн голосование) ИЛИ Aragon DAO на Polygon **Amoy testnet**.
- Safe multisig (тестовый): стартовая конфигурация **5 хранителей, 3 подписи из 5**. Основатель — обычный хранитель.

**Этап 5 — Смарт-контракты (цифровое воплощение конституции), TESTNET ТОЛЬКО:**
- Казна, голосование, распределение вознаграждений, репутация, аудит, анти-злоупотребление.
- Поэтапные выплаты, лимиты, коллективная проверка, апелляции — по конституции.
- ОБЯЗАТЕЛЬНО: тесты + security-review перед любым упоминанием mainnet. Реальные средства — только после явного одобрения и аудита.

**Этап 6 — AI-агенты (служат миссии, НЕ владеют, НЕ имеют власти):**

# ПРИЛОЖЕНИЕ: ПОЛНЫЙ МАНИФЕСТ / КОНСТИТУЦИЯ (north star и закон проекта)

# PUBLIC TRUST DAO v1.0
## Единый манифест, конституция, цели, блокчейн-архитектура и техническое задание

# Миссия
Создать открытую, децентрализованную систему взаимной поддержки и общественных благ, обеспечивающую людям базовую безопасность, человеческое достоинство, жильё, питание, медицинскую помощь, образование, восстановление самостоятельности и возможность помогать другим людям и обществу.
Проект не является инвестиционной схемой, финансовой пирамидой или инструментом извлечения прибыли.
Проект представляет собой открытую инфраструктуру взаимопомощи, управляемую конституцией, прозрачной блокчейн-казной, открытым кодом, искусственным интеллектом и сообществом.

# Главный принцип
Никто не является владельцем проекта. Никто не является постоянным бенефициаром. Основатель является обычным участником. Проект принадлежит своей миссии, конституции и сообществу. Люди и ИИ являются временными хранителями этой миссии.

# Блокчейн-основание проекта
Public Trust DAO — блокчейн-нативная система общественных благ на публичных сетях и открытых протоколах. На начальном этапе свой блокчейн не создаётся. Используются: Polygon; Ethereum; другие открытые сети по мере необходимости.

# Принципы блокчейн-архитектуры
открытость; прозрачность; неизменяемость записей; отсутствие центрального владельца; распределённое управление; проверяемость; открытый код; совместимость с другими протоколами.

# Высшая цель
Устойчивая мировая сеть взаимопомощи, в которой человек, получивший поддержку, со временем получает возможность помогать другим.

# Основополагающие принципы
человеческое достоинство; сострадание; взаимопомощь; прозрачность; добровольность; отсутствие эксплуатации; отсутствие единоличной власти; общественная польза; открытый код; защита от злоупотреблений; справедливость; долгосрочная устойчивость; развитие самостоятельности; приоритет реальной пользы.

# Основные направления деятельности
Жильё (предотвращение бездомности, аренда, ЖКХ, временное/экстренное жильё, переезд, кооперативное/общественное жильё, строительство). Питание (продукты, горячее питание, доставка, дети, семьи, пожилые). Здоровье (лечение, лекарства, стоматология, психология, диагностика, реабилитация, инвалидность, паллиатив, экстренная мед.помощь). Поддержка семей (дети, беременные, одинокие/многодетные родители, потеря кормильца). Пожилые (уход, сопровождение, доставка, быт, оплата нужд). Образование (переквалификация, профобучение, языки, цифровая грамотность, открытые знания, таланты). Трудоустройство (поиск работы, документы, самозанятость, инструменты, рабочие места, кооперативы). Экстренная помощь (потеря жилья/работы, гум.кризисы, бедствия, ЧС). Животные (лечение, кормление, приюты, спасение). Социальная интеграция (одинокие, мигранты, восстановление связей, после тяжёлых ситуаций). Общественные блага (open source, наука, образование, экология, цифровая инфраструктура, свободные знания). Поддержка участников фонда — на равных основаниях.

# Вознаграждение за общественно полезную деятельность
помощь пожилым/детям, доставка продуктов, ремонт жилья, волонтёрство, обучение, развитие проекта, программирование, юр./мед. помощь, модерация, помощь животным, сопровождение нуждающихся, образовательные материалы. **Вознаграждение никогда не зависит от привлечения новых участников.**

# Конституционные запреты
Запрещается: обещать доходность; гарантировать прибыль; создавать финансовые пирамиды; платить за привлечение людей; концентрировать власть; скрывать операции; использовать фонд в личных интересах; коррупция; фаворитизм; дискриминация.

# Роли
Нуждающийся; Участник; Волонтёр; Доверенный участник; Хранитель; Наблюдатель; AI Guardian.

# Репутационная система
Учитываются: время участия; выполненные задачи; помощь другим; отсутствие нарушений; рекомендации сообщества; длительность участия.

# Антизлоупотребление
поэтапные выплаты; лимиты; коллективная проверка; публичные отчёты; аудит; апелляции; репутационные санкции; временные ограничения; исключение участников. Система должна делать честность выгоднее мошенничества.

# Конституция человеческого достоинства
Каждый человек обладает достоинством независимо от: дохода; происхождения; образования; возраста; национальности; состояния здоровья; социального положения. Помощь должна увеличивать самостоятельность человека, а не зависимость.

# Конституция справедливого распределения (приоритет)
1. Угроза жизни. 2. Угроза потери жилья. 3. Медицинская помощь. 4. Питание. 5. Дети. 6. Пожилые. 7. Инвалидность. 8. Восстановление самостоятельности. 9. Образование. 10. Долгосрочная общественная польза.

# Общественная казна
Средства существуют в блокчейне. Используются: Safe multisig; смарт-контракты; DAO-модули; публичные реестры. Ни один человек не обладает единоличным контролем. Стартовая конфигурация: 5 хранителей; 3 подписи из 5. Основатель имеет те же права, что остальные хранители.

# Bootstrap-период
Временный bootstrap-кошелёк — только для запуска инфраструктуры. Bootstrap-хранитель не владелец. После формирования сообщества средства переводятся в общественную казну.

# Смарт-контракты
Цифровое воплощение конституции: хранение средств; голосование; управление казной; регистрация решений; распределение вознаграждений; репутация; аудит; защита от злоупотреблений.

# AI-агенты
Guardian / Audit / Fairness / Reputation / Housing / Governance / Mediator / Documentation AI. ИИ не обладает властью, не владеет средствами, служит миссии и помогает соблюдать конституцию.

# Хранение данных
GitHub; GitHub Pages; IPFS; распределённые хранилища.

# Технологическая инфраструктура
GitHub; GitHub Pages; Cloudflare Pages; MetaMask; Safe; Polygon; Ethereum; IPFS; Snapshot; Aragon; OpenAI; Claude; локальные модели.

# Публичность
Все документы открыты. Все решения объясняются. Все операции публикуются. Все изменения фиксируются.

# Юридический статус
Этап 1 — Open Source Initiative. Далее возможно: некоммерческий фонд; кооператив; Public Benefit Foundation; DAO.

# Конечная цель
Глобальный Public Trust DAO — общественный цифровой траст, где ресурсы принадлежат не отдельным людям, а миссии, конституции и сообществу. Человек, получивший помощь, по возможности становится источником помощи для других. Главные ценности: человеческое достоинство; взаимопомощь; справедливость; общественное благо; способность общества заботиться о своих членах. Ни человек, ни организация, ни государство, ни ИИ не являются владельцами проекта. Проект принадлежит своей миссии, конституции и сообществу.
