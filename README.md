# Public Trust DAO

**Открытая, децентрализованная система взаимопомощи и общественных благ.**

Public Trust DAO обеспечивает людям базовую безопасность, человеческое достоинство, жильё, питание, медицинскую помощь, образование и восстановление самостоятельности — и возможность, получив помощь, со временем помогать другим.

> ⚠️ **Это НЕ инвестиция, НЕ финансовая пирамида, НЕ инструмент извлечения прибыли.** Это открытая инфраструктура взаимопомощи, управляемая конституцией, прозрачной блокчейн-казной, открытым кодом и сообществом.

## Главный принцип
Никто не является владельцем проекта. Основатель — обычный участник. Проект принадлежит своей **миссии, конституции и сообществу**. Люди и ИИ — временные хранители миссии.

- `contracts/` — смарт-контракты (TESTNET first).
- `web/` — публичный сайт (GitHub Pages).
- `governance/` — управление (Safe multisig 3-из-5, Snapshot/Aragon).
  - [`governance/registry/`](governance/registry/) — публичный, проверяемый,
    защищённый от подмены реестр решений (append-only hash-chain).
  - [`governance/safe/`](governance/safe/) — макет тестовой казны Safe multisig
    3-из-5 (testnet, без реальных денег/ключей) + валидатор рельс.
  - [`governance/snapshot/`](governance/snapshot/) — макет off-chain голосования
    Snapshot (1 человек = 1 голос, не плутократия) + валидатор рельс.
- `ai-agents/` — AI-модули, служащие миссии (не владеют средствами, не имеют власти).
